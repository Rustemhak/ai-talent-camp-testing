Система оценки

Три измерения качества продукта

🎨 UX / User Experience 40%

Навигация, дизайн, отзывчивость, читаемость, обработка ошибок

⚙️ Functionality 40%

Работа фич, соответствие документации, стабильность, edge cases

⚡ Performance 20%

Время загрузки < 3с, отклик < 500мс, стабильность, ресурсы

overall_score = ux * 0.4 + functionality * 0.4 + performance * 0.2

Как Claude тестирует

1

Открывает URL продукта

Playwright MCP запускает браузер

2

Делает скриншот

Анализирует визуальное состояние

3

Исследует интерфейс

Кликает, вводит текст, навигирует

4

Фиксирует находки

Баги, проблемы UX, фичи

5

Выставляет оценки
